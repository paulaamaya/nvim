# nvim
My neovim config.

## Core Keymaps

**Insert Mode**
- `jk` Exit insert mode

**Normal Mode**
- `+` Increment number.
- `-` Decrement number.
- `sv` Split window vertically.
- `sh` Split window horizontally.
- `se` Make splits equal size.
- `sx` Close current split window.
- `to` Open new tab.
- `tx` Close current tab.
- `tn` Go to next tab.
- `tp` Go to previous tab.
- `tf` Open current buffer in new tab.
- `U` Redo last change.
